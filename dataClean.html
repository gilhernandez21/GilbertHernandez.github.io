SQL: Batch Data Cleanup

  • Let's start with querying the data that has missing values, so we know what needs to be updated.
  • Now let’s export the data to an excel sheet so we can inspect it.
  • Looking at the spreadsheet, it looks like we need to fill in column D with the values from column E.
  • We need to be sure to set conditions to make it as specific as possible so in this case we’ll condition it with report journal ID, Well ID and Wellbore ID.
  • Now we can add the following formula to the first row and copy it down all the way to the last row.
="update DM_Well_Plan SET estimated_duration = "&D2&" where well_id = '"&F2&"' and report_journal_id = '"&E2&"' and wellbore_id = '"&G2&"';"
  • Now we’re ready to copy and paste all of the Update statements(column H) into sql developer.
  • Lastly, click on "Run All" to run the entire page of scripts at once.
  • That's it! All the rows were corrected at the same time instead of having to go Well by Well.
